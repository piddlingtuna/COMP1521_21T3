1 == Negative

## Two's Complement

0    0    0    0    0    0    0    0
-128 64   32   16   8    4    2    1
-2^7 2^6  2^5  2^4  2^3  2^2  2^1  2^0

To convert from + to -, bitwise not and + 1. Works both ways!

6    == 0000 0110
~6   == 1111 1001
~6+1 == 1111 1010


0    == 0000 0000
~0   == 1111 1111
~0+1 == 0000 0000
-0   == 0000 0000
